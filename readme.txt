You will reuqire python 2.7 to run the program.

First run the process_raw_data.py. This process the raw data and convert into required format.
Now run the adaboost.py. This will execute the algorithm. 
It will take some time to completely execute the program.
Algorithm halts after 57 rounds and once AUC is greater than 0.985.
